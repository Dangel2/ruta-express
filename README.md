   

#  Ruta Express

Ruta Express es una plataforma web de mandados y entregas desarrollada con React, Node.js y PostgreSQL.

## Características

- Registro e inicio de sesión de usuarios.
- Creación y seguimiento de pedidos.
- Dashboard administrativo.
- Gestión de promociones.
- Estados de pedidos:
  - Pendiente
  - En camino
  - Entregado
- Base de datos PostgreSQL.
- API REST con Express.
- Autenticación JWT.

## Tecnologías

### Frontend
- React
- Vite
- React Router
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express
- PostgreSQL
- JWT
- CORS

## Instalación

### Clonar repositorio

```bash
git clone https://github.com/TU-USUARIO/ruta-express.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Variables de entorno

Crear un archivo `.env` en la carpeta backend:

```env
PORT=5000
DATABASE_URL=TU_DATABASE_URL
JWT_SECRET=TU_CLAVE_SECRETA
```
